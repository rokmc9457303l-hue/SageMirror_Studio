# 🎬 캡컷 프로젝트 전용 저장소 (CapCut Projects)

이 폴더는 **현자의 거울 스튜디오**와 **캡컷(CapCut)**이 하나로 연결되는 전용 작업 공간입니다.

### ⚙️ 필수 세팅 방법
1. 캡컷(CapCut) 실행
2. 우측 상단 톱니바퀴 ➡️ 설정(Settings) ➡️ 프로젝트(Project) 탭
3. **'저장 위치(Save to)'**를 이 폴더(`C:\SageMirror_Production\05_CapCut_Projects`)로 지정하세요!

이제 캡컷에서 프로젝트를 생성하면, 이곳에 프로젝트 파일들이 모이게 되며 조립기가 1초 만에 파일들을 찾아냅니다.
