# 🚀 AI-Project: 통합 지식 창고

본 보관소는 흩어져 있던 옵시디언 자료들을 하나로 통합한 지식 창고입니다.

## 📂 폴더 구조 안내
- **`00_System`**: 보관소 가이드 및 시스템 설정
- **`01_Articles`**: 핵심 지식 메모 및 프롬프트 자료
- **`02_Daily`**: 날짜별 일기 및 업무 기록

## 💡 시작하기
1. **Obsidian**을 실행합니다.
2. `Open folder as vault`를 선택합니다.
3. `C:\Users\admin\Downloads\AI-Project` 폴더를 선택하세요.
