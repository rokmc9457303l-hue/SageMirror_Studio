# [[The Theory of the Leisure Class (유한계급론)]]

## 📌 Brief Summary
경제학자 소스타인 베블런(Thorstein Veblen)이 1899년에 발표한 사회학의 고전으로, 부유한 계층이 자신의 지위를 과시하기 위해 행하는 '과시적 소비(Conspicuous Consumption)'와 '과시적 여가'의 본질을 날카롭게 해부한 책입니다.

## 📖 Core Content
- **과시적 소비:** 물건의 실질적인 유용성보다, 그것을 소유함으로써 자신이 노동하지 않아도 될 만큼 부유하다는 것을 남들에게 보여주려는 심리가 소비의 핵심임을 분석합니다.
- **유한계급(Leisure Class):** 생산적인 노동을 면제받고 여가를 즐기는 것을 최고의 사회적 지위로 여기는 계층의 속성과 그들이 만들어내는 문화적 허영심을 묘사합니다.
- **모방 소비:** 하층 계급이 상층 계급의 소비 양식을 모방하려 함으로써 사회 전체적으로 비생산적인 낭비가 발생하는 구조를 설명합니다.
