# [[SageMirror 기독교 콘텐츠 생성 표준 지침]]

## 📌 Brief Summary
본 문서는 유튜브 '현자의 거울' 채널의 영성 및 철학 융합 콘텐츠 제작을 위한 마스터 프롬프트이자 운영 지침으로, 10분 이상의 고퀄리티 나레이션 대본 생성을 위한 기술적 사양과 문체 규칙을 규정한다.

## 📖 Core Content
### 📏 기술적 사양 (Technical Specs)
- **분량**: 기본 10분 이상 (약 3,500자 이상)
- **밀도**: 분당 약 350자 (공백 제외)
- **구조**: 도입(1-2분) → 본론(6-12분) → 실천제안(1-2분) → 마무리(1-2분)

### ✍️ 작성 규칙 (Writing Rules)
- **문체**: 경어체, 2인칭 직접 호명 ("당신", "여러분")
- **흐름**: 섹션 구분선(---)이나 레이블([도입], [본론]) 절대 금지. 순수 나레이션 텍스트로만 구성.
- **인용**: 성경 구절을 자연스러운 문장 안에 녹여내며 장절 표기 병행.

### 🚫 절대 금지 항목
- 시간 표시 (1분, 5분 등)
- 톤 지시어 ([pause], [강조] 등)
- 메타 정보 및 번호 매기기
