# [[나레이션 오디오 자동화 마스터 설계도]]

## 📌 Brief Summary
본 문서는 스튜디오 파이프라인에서 생성된 '순수 텍스트 대본'을 바탕으로, Qwen 3 (또는 유사 TTS) 엔진을 제어하여 고품질 50대 남성 앵커 톤의 나레이션을 생성하고, 로컬 오디오 파일을 캡컷(CapCut)과 자동 동기화(Sync)하기 위한 마스터 가이드입니다.

## 📖 Core Content

### ◼️ [Node-01] 내부 대본 로드 (Pure Script Fetch)
*   **목적:** 오직 물리적 호흡(띄어쓰기/줄바꿈)만 남은 순수 텍스트 대본을 불러옵니다.
*   **방식:** 112개(또는 유동적인 개수)의 씬 대본이 순번(ID)대로 쪼개져 시스템에 로드됩니다.

### ◼️ [Node-02] Qwen 3 마스터 프롬프트 생성기 (Voice Persona Guide)
*   **목적:** 나레이션 생성 엔진(Qwen 3)이 전체 영상을 하나의 일관된 철학적 톤앤매너로 연기하도록 최면 프롬프트를 부여합니다.
*   **Qwen 3 복붙용 강제 프롬프트:**
    > "너는 지금부터 깊은 철학적 사유를 전달하는 베테랑 다큐멘터리 내레이터야. 나이는 50대 후반의 남성이고, 가슴을 울리는 중후하고 평온한 중저음(Baritone)의 목소리를 가졌어. 아래 대본을 읽을 때 감정을 과장하지 마. 뉴스를 진행하는 앵커처럼 발음은 아주 정확하되, 일반 뉴스보다 1.2배 느린 속도로 천천히, 단어의 무게감을 곱씹듯이 관조적이고 차분한 톤앤매너를 유지해 줘. 대본에 있는 띄어쓰기와 줄바꿈은 네가 숨을 쉬는 여백(Pause)이야. 오디오 시작과 끝에는 0.5초의 무음을 꼭 넣어줘.\n\n**대본:** [순수 대본]"

### ◼️ [Node-03] 로컬 오디오 폴더 자동 매칭 (External Folder Sync)
*   **목적:** 사용자가 생성해 온 실제 '나레이션(목소리) 음원 파일'들을 대본 ID와 정확히 1:1로 짝지어줍니다.
*   **방식:** 지정된 폴더(`C:/Tubi_Audio_Export/`) 내부를 스캔하여, 대본 번호와 일치하는 오디오 파일(예: `001_Audio.wav`)이 있는지 확인합니다. 파일이 존재하면 UI에 `[✅ 완료]`를 띄우고, 없으면 `[❌ 누락]`을 띄워 휴먼 에러를 방지합니다.

### ◼️ [Node-04] 캡컷 자동배치 마스터 JSON 연동 (Ready for Handoff)
*   **목적:** 모든 오디오 파일의 매칭이 `✅ 완료`되면, 해당 경로와 텍스트를 잠금(Lock) 처리하여 `Audio_Sync_Master.json` 파일로 구워냅니다. 
*   **결과:** 이후 [캡컷 자동 조립기]가 이 JSON을 읽고 마법처럼 타임라인에 오디오를 정확한 위치에 꽂아 넣습니다.
