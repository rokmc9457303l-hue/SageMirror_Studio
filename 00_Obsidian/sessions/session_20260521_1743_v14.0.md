## 📊 파트별 구현 상세

### 파트 6 — 나레이션 & 배경음악 (Opal Dispatch)

**데이터 입력**: `p34_narration_script` (나레이션 대본) 또는 `p34_image_script` (이미지 대본)

**핵심 기능**:
- 씬 번호(`001`~`112`) + 나레이션 텍스트 자동 파싱
- 112개 씬을 8개 계정(1번계정~8번계정)에 **14씬씩** 균등 배분
- Gemma AI 배분 결과 정규식 검증: `r"^(\d+번계정)\s*\|\s*(\d{3})\s*\|\s*(.+?)\s*\|\s*@(.+?)@\s*$"`
- 배분 실패 줄 자동 리스트업 + 재생성 버튼
- `p6_opal_data` (dict list) 세션 저장 (JSON 직렬화 안전)
- CSV 다운로드 (`utf-8-sig`, Excel 한글 깨짐 방지)
- 옵시디언 자동 백업 + Git Push

**출력**: `st.session_state.p6_opal_data` → 파트 7로 전달

---

### 파트 7 — 숏폼 생성 CapCut Bridge

**데이터 입력**: `p6_opal_data` (파트 6 배분 결과)

**핵심 기능**:
- 씬별 자막 지속시간 자동 연산: **글자당 0.25초, 최소 4.0초**
- 파일명 자동 생성 규칙:
  - 이미지: `scene_001.png` ~ `scene_112.png`
  - 비디오: `video_001.mp4` ~ `video_112.mp4`
  - 나레이션: `narration_001.mp3` ~ `narration_112.mp3`
- CapCut Bridge DataFrame 컬럼: `씬번호`, `계정`, `이미지파일`, `비디오파일`, `나레이션파일`, `자막텍스트`, `지속시간(초)`, `나레이션(@Protagonist)`
- `p7_capcut_data_v2` 세션 저장 (dict list)
- CSV 다운로드 (`utf-8-sig`)

**출력**: `st.session_state.p7_capcut_data_v2` → 파트 8로 전달

---

### 파트 8 — 캡컷 최종 조립 Dashboard

**데이터 입력**: `p7_capcut_data_v2` (파트 7 CapCut 데이터)

**핵심 기능**:
- `C:\SageMirror_Production\06_Video_Clips` 폴더 실시간 스캔
- `video_001.mp4` ~ `video_112.mp4` 파일 존재 여부 매칭률(%) 계산
- `st.progress` 바 시각화 + 매칭 완료/누락 파일 리스트업
- 통계 카드: 총 씬 수 / 총 예상 재생시간 / 총 자막 글자 수
- 옵시디언 자동 백업 + Git Push
