# Sage Mirror Studio Session — v16.1.1
날짜: 2026-05-26 06:56

## 변경 사항 요약
- **Gemma Assistant 제작 흐름 기억형 Assistant 강화**:
  - `rag_memory_utils.py`에 `build_recent_activity_memory()` 함수를 개발하여 에피소드 정보, 진행 중인 파트 정보, 최근 References 파일 목록, 최근 검색 키워드 및 대화 질문들을 6000자 이내로 요약해 주입하도록 설계했습니다.
  - `sage_popups.py` 내의 챗 대화 처리 흐름인 `popup_assistant` 내부에서 `[현재 파트 컨텍스트] -> [RECENT_ACTIVITY_MEMORY] -> [REFERENCES_MEMORY]` 순서로 결합되도록 구성하여 대화의 흐름 인지력을 대폭 강화했습니다.
  - `save_workspace_state` 임포트 버전을 `app_v16_1_1`로 일치시켜 버전 불일치로 인한 오동작 현상을 방지했습니다.
  - `RUN_APP.bat`, `RUN_DEBUG.bat`, `RUN_APP.vbs` 배치 및 스크립트 실행 대상을 `app_v16_1_1.py`로 갱신하여 신규 버전 정식 릴리즈를 준비했습니다.

## 수정된 함수 목록
- `rag_memory_utils.py` : `build_recent_activity_memory(state_dict: dict, max_chars: int = 6000) -> str` (신규 추가)
