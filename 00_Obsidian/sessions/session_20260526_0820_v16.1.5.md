## 🛠️ 수정 및 추가된 함수 목록
- **`research_router.py`** [NEW]
  - `should_trigger_research(text: str) -> str | None`: `[NEED_RESEARCH: 키워드]` 감지 및 정제
  - `run_tavily_research(query: str, api_key: str, max_results: int = 5) -> dict`: Tavily 검색 API 호출 및 에러 핸들링
  - `clean_search_query(query: str) -> str`: 검색어 특수 문자 및 개행 공백 클렌징
  - `format_search_results_markdown(web_results: list) -> str`: Tavily 검색 데이터를 마크다운 문자열로 변환
  - `build_tavily_rag_context_core(search_history: list) -> str`: 젬마 RAG 컨텍스트 생성을 위한 인터넷 리서치 이력 빌드
  - `build_gemini_search_context_core(grounding_results: list, summary: str = "") -> str`: Gemini 검색 그라운딩 결과의 RAG 컨텍스트화
  - `build_youtube_search_context_core(yt_results: list) -> str`: 유튜브 검색 결과 RAG 컨텍스트 빌드
  - `build_research_markdown_document(query, obs_results, web_search_text, final_summary, model_name, part_key, part_label, classification_md) -> str`: 최종 파트 RAG 검색 결과 저장용 마크다운 산출물 생성
  - `format_source_citation(url: str, title: str, author: str, date_str: str) -> str`: 출처 태그 포맷 표준화
