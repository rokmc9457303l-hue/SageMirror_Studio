# Sage Mirror Studio Session — v16.1.7 (2026-05-26 08:53)

## 📌 변경 사항 요약
- **Agent Registry System 도입**:
  - Gemma Assistant가 사용할 수 있는 에이전트 도구들의 메타데이터(설명, 별칭, 패턴, 카테고리, 활성화 여부 등)를 통합 관리하는 중앙 레지스트리 모듈 `agent_registry.py`를 신규 설계하여 추가했습니다.
  - `agent_registry.py`는 순수 Python 모듈로서 Streamlit UI 및 플랫폼 의존성을 원천 배제하여 구조적 결합력을 높였습니다.
  - `agent_toolkit.py`를 수정하여 기존에 하드코딩되어 있던 툴 목록 사전을 제거하고, `agent_registry`를 import하여 툴 목록 및 패턴 정합성을 동적으로 빌드하도록 리팩토링했습니다.
  - `sage_popups.py`에서 `agent_toolkit`을 통해 정규화 및 탐색할 때 Registry 사전에 등록되어 비활성화(`enabled=False`) 처리된 도구들은 자동으로 감지 대상에서 배제되도록 정합성을 고도화했습니다.
  - 신규 버전 `app_v16_1_7.py`를 제작하여 버전 명을 `v16.1.7`로 갱신하고, 기존 동작 성능 및 안정성을 보존했습니다.
  - 배포 스크립트 3종(`RUN_APP.bat`, `RUN_DEBUG.bat`, `RUN_APP.vbs`)의 Streamlit 기동 파일 실행 대상을 `app_v16_1_7.py` 및 `8505` 포트로 갱신했습니다.

## 🛠️ 수정 및 추가된 함수 목록
- **`agent_registry.py`** [NEW]
  - `get_agent_tool_registry() -> dict`: 중앙 에이전트 도구 레지스트리 반환

  - `get_enabled_tools() -> list[str]`: 활성화 상태인 도구 명칭의 목록 반환

  - `get_tool_description(tool_name: str) -> str`: 툴의 설명 반환
  - `get_tool_category(tool_name: str) -> str`: 툴의 카테고리 반환
- **`agent_toolkit.py`** [MODIFY]
  - `get_supported_agent_tools() -> dict`: 레지스트리로부터 `enabled=True`인 도구들의 이름 및 패턴 정보를 동적으로 반환하도록 수정
  - `normalize_tool_name(name: str) -> str`: 레지스트리 내 정규화 함수로 위임
- **`app_v16_1_7.py`** [NEW]
  - 상단 헤더 버전을 `v16.1.7`로 갱신하고 동작 안정성 유지
