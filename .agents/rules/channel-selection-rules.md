# 현자의 거울 — 유튜브 타겟 채널 선정 규칙서
## 어떤 AI든 이 규칙을 반드시 따를 것

## 1. 절대 조건 (위반 시 즉시 제외)
- 재사용/카피 콘텐츠 채널 → 제외
- AI 자동화/페이스리스 채널 → 제외
- 단순 텍스트+음악 조합 채널 → 제외

## 2. 우선순위
- 1순위: 사람이 직접 얼굴 출연하는 채널
- 2순위: 오리지널 AI 채널 (재사용 없음)

## 3. 핵심 선정 기준
- 타겟: 40-70대 (4070세대)
- 주제: 심리학, 철학, 위로, 인문학, 성경적 지혜
- 구독자 수: 적을수록 좋음 (숨은 명채널)
- 조회수: 구독자 대비 압도적으로 높은 채널
- 국내+국외 모두 탐색

## 4. 출력 형식
채널명 | 구독자수 | 대표영상 조회수 | 채널URL | 선정이유
